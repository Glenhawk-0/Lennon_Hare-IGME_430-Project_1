Search Books|/api/books|Get/Head

Query Params

title: (optional) - A Book's name to filter by

author: (optional) - An author's name to filter by

year: (optional) - A release year to filter by

Returns

A JSON Array of book Objects

Format: [{title: titleString1, author: authorString1, year: number1 }, {title: titleString2, author: authorString2, year: number2 }...]

Example

Request:/api/books?title=title1&author=author1&year=2000

result: [{title: 'title1', author: 'author1', year: 2000}]

Book Titles|/api/books/titles|Get/Head

Query Params

author: (optional) - An author's name to filter by

year: (optional) - A release year to filter by

Returns

A JSON Array of book objects with only titles

Format: [{title: string1},{title: string2},{title: string3 },...]

Example

Request:/api/books/titles?author='butt head'&year=1800

result: [{title: 'Butt Head the force awakens'}, {title: 'butt head the return of the butt'}]

Books By Page Count|/api/books/pages|Get/Head

Query Params

Min Pages: (optional*) - a minimum page number to filter by

Max Pages: (optional*) - a maximum page number to filter by

(at least one needs to have an input)

Returns

A JSON Array of book Objects, or a 404 if it lacks both a min and max

Format: [{title: string1, pages: num1},{title: string2, pages: num2},...]

Example

Request:/api/books/pages?min=100&max=186

result: [{{title: 'this is a short book', pages: 101}, {title: 'this is a tad less short', pages: 164}}]

Add A Book|/api/books/add|Post

Supported Body Formats

application/json

application/x--www-form-urlencoded

Body Params

title Type: string - name of the Book

author Type: string - the name of the author

year Type: number - The year of the books release

Returns

A JSON Book object representing the new book

Format: [{title: titleString, author: authorString, year: number }]

Example

Request:/api/books/add

Body: [{title: 'myBook', author: 'me', year: 2025 }]

result: [{title: 'myBook', author: 'me', year: 2025 }]

Update A Book|/api/books/update|Post

Supported Body Formats

application/json

application/x--www-form-urlencoded

Query Params

title: (required) - A Book's name to filter by

Body Params

author Type: string - the name of the author

year Type: number - The year of the books release

Returns

A JSON Book object representing the updated book or a 404 if their is no title given

Format: [{title: titleString, author: authorString, year: number }]

Example

Request:/api/books/update

Body: [{title: 'thatBook', author: 'notThem', year: number 2026 }]

result: [{title: 'thatBook', author: 'notThem', year: number 2026 }]
